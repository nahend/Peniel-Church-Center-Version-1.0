# Peniel Church Center v1.0

Production monorepo for Peniel Ethiopian Evangelical Church: bilingual public website and authenticated Church Center SaaS dashboard.

## Apps
- `apps/website`: public bilingual website.
- `apps/church-center`: authenticated admin and operations console.

## Packages
Shared packages provide UI, auth, database metadata, workflows, notifications, forms, types, and utilities.

## Commands
```bash
pnpm install
pnpm dev
pnpm build
pnpm lint
pnpm test
```
